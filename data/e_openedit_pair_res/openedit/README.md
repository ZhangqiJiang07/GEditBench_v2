# OpenEdit Benchmark Evaluation Buckets

This folder groups pairwise evaluation outputs by evaluation dimension for the GEditBench v2/OpenEdit benchmark.

Subdirectories

- `eval_if_metadata/`: instruction-following comparisons.
- `eval_vc_metadata/`: visual-consistency comparisons.
- `eval_vq_metadata/`: visual-quality comparisons.

Each subdirectory stores timestamped JSONL result files.
