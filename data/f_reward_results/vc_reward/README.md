# VCReward Benchmark Results

This folder groups evaluation outputs for the `vc_reward` benchmark.

Subdirectories

- `eval_vc/`: default visual-consistency evaluation pipeline.
- `eval_vc_gemini/`: Gemini-based ablation results.
- `eval_vc_gpt5d1/`: GPT-based ablation results.
- `eval_vc_qwen3/`: Qwen-based ablation results.

Each subdirectory stores timestamped JSONL files with one evaluated pair per line.
