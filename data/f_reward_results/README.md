# Reward Benchmark Evaluation Results

This folder stores evaluation outputs for reward-model benchmarks such as `vc_reward`.

Typical contents

- `vc_reward/`: benchmark-specific result folders.
- `.cache/`: resumable evaluation caches.

Canonical output pattern

- `vc_reward/eval_vc/<timestamp>.jsonl`
- `vc_reward/eval_vc_gemini/<timestamp>.jsonl`
- `vc_reward/eval_vc_gpt5d1/<timestamp>.jsonl`
- `vc_reward/eval_vc_qwen3/<timestamp>.jsonl`
